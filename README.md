# ClasseObjetoJava

Este projeto contém exercícios de **Programação Orientada a Objetos em Java**, desenvolvidos para a disciplina de Paradigmas de Programação I.  
O objetivo é praticar conceitos como **classes**, **objetos**, **atributos**, **métodos** e **construtores**.

## 📌 Exercícios incluídos

### 🧮 Exercício 1 — Conta Bancária  
Simula o funcionamento de uma conta bancária, permitindo depósitos, saques e encerramento de conta, com validação de saldo.

### 📺 Exercício 2 — Sistema de Streaming  
Modelo de sistema de assinatura de streaming que controla o usuário, plano, mensalidade e filmes assistidos.

### 🏀 Exercício 3 — Placar de Basquete  
Registra pontos de dois times e acompanha os períodos do jogo.

## 📁 Estrutura do projeto

- `Conta.java` — Classe que representa conta bancária  
- `TestaConta.java` — Classe de teste para Conta  
- `Streaming.java` — Classe que representa serviço de streaming  
- `TesteStreaming.java` — Classe de teste para Streaming  
- `Placar.java` — Classe que representa placar de basquete  
- `TestePlacar.java` — Classe de teste para Placar

## 🚀 Como executar

Para rodar os testes, basta compilar e executar cada classe com o método `main`, por exemplo:

```bash
javac Conta.java TestaConta.java
java TestaConta
